# USEME

## Start Server
```
./server
```

## WRITE
```
./rfs WRITE <local_file> <remote_path>
```

## GET (latest)
```
./rfs GET <remote_path> <local_file>
```

## GET (specific version)
```
./rfs GET <remote_path> <local_file> <version_number>
```

## LS
```
./rfs LS <remote_path>
```

## RM
```
./rfs RM <remote_path>
```

## STOP
```
./rfs STOP
```
